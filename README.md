# Portfolio — Mohamed Ouijjane

Professional personal portfolio built with **Next.js, TypeScript, and Tailwind CSS**.

## Purpose

This repository contains a modern, production-ready portfolio designed to present projects, skills, and experience in a clear and professional way.

The focus is on:

- Clean architecture
- Maintainable code
- Clear presentation of work
- Long-term scalability

## Featured Work

- **Portfolio Platform** — Modern Next.js App Router website with MDX-based content
- **BI / Data Analytics** — ETL pipelines, star schema modeling, and dashboards
- **Data Mining** — Practical analysis using clustering and association rules

## Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- MDX / Markdown

## Local development

```bash
npm install
npm run dev
```
